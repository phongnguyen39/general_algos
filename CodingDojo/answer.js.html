    Quick Quiz

  1. How many inputs are accepted by the == operator and the === operator, respectively?

  2. Do inputs to == and === operators need to be the same data type, for the operators to function?

  3. What is the data type of the output value produced by the == and === operators?

    Answers

  1. Two inputs for both == and === operators. These operators compares variables that reside on both sides of the equal sign.

  2. For ==, the data types do not need to match. JS will convert data types as needed. They are loosely compared. For ===, the data types must match.

  3. == and === outputs are the boolean datatype, but === requires the two variables' being compared must match.
